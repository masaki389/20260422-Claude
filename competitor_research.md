## 伸びるタイトルの共通法則（7要素）

| 要素 | 具体例 |
|------|--------|
| ①具体的な数字 | 月収15万・年金6万・4万円→460万・3年後 |
| ②屈辱・見下し | 笑われた・恥ずかしい・貧乏人・まだ働いてるの？ |
| ③時間経過で逆転 | 3年後・5年後・10年後・1ヶ月後 |
| ④人間関係の痛み | 娘・息子夫婦・姑・親友・夫 |
| ⑤生活のリアル | パート・スーパー・団地・食費・家賃 |
| ⑥制度・お金の不安 | 年金受給・貯金ゼロ・老後2000万 |
| ⑦末路 or スカッと | 崩壊・末路・逆転・青ざめた・腰を抜かした |

**最強は3つ以上が重なった時。**

## 勝ちパターン公式

```
【具体的金額 or 状況】主人公への屈辱（笑われた・言われた）→「〇年後」逆転の予感
```

## 幸子チャンネルで弱かったタイトルの共通点

- 「心を守る方法」「再生録」「老後の生活」など抽象ワード中心
- テーマだけ立派で"事件"が弱い
- 金額・屈辱・逆転のどれかが欠けている

視聴者は"学び"より先に「え、その後どうなったの？」でクリックする。
