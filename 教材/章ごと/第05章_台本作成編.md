# 第5章：台本作成編
### 再生数を左右する台本の細部

---

AIアニメの制作で一番重要なのは、実は台本だと思っています。

どれだけ画像生成が上手くなっても、台本が弱ければ動画は伸びない。逆に台本がしっかりしていれば、画像の品質が多少ばらついていても視聴者は見続けてくれます。

この章では、実際に失敗した台本と改善後の台本を比較しながら、「何が再生数を左右するのか」を話します。

---

## 落とし穴①：冒頭のつかみが弱い

### 実際の失敗例

山崎製パンの転職紹介動画の最初の台本案、冒頭はこうでした。

> 「日本で売られるパンの約4分の1は、山崎製パン製です。」

読んで「だから何？」と思いましたよね。視聴者も同じことを感じます。

この冒頭で何人が離脱するかを考えると、使えません。

---

### なぜ弱いのか

冒頭の役割は「この動画を最後まで見る理由を作ること」です。

「日本のパンの4分の1が山崎製パン」という情報は確かに事実ですが、それを知ったところで「続きを見たい」という気持ちにはならない。情報を提供しているだけで、視聴者の感情を動かしていない。

---

### 解決策：逆説・矛盾から始める

改善後の冒頭はこうしました。

> 「残業は45時間を超える。有給も取りにくい。最悪12連勤になることもある。
> それでも、平均勤続年数は16年。離職率は7%。
> その会社が、山崎製パンだ。」

「こんなにブラックなのに、なぜ人が辞めないのか？」という疑問を視聴者の頭に作ることができる。

続きを見たくなる。

冒頭の構造として有効なのは、「一般的に知られていること」と「実際の姿」の矛盾を見せることです。

```
一般的なイメージ：「キツそう、ブラックそう」
実際のデータ：「勤続16年、離職率7%」
↓
「なぜ？」という疑問が生まれる
↓
続きを見る
```

転職系の動画に限らず、幸子チャンネルのドラマでも同じ構造が使えます。「こうなりそう」と思わせておいて「実はこうだった」という展開を最初に示唆することで、視聴者を引き込む。

---

## 落とし穴②：数字を言い切らない

### 実際の失敗例

```
「ボーナスは多いと言われています」
「退職金もかなりの額がもらえるようです」
「社宅の家賃は格安で住めます」
```

これ、全部曖昧です。

視聴者がこの動画を見に来た理由は「具体的な数字を知りたい」からです。なんとなくの情報なら別にわざわざ動画を見なくていい。

---

### 解決策：断定する

```
「ボーナスは年2回。合計約100万円。」
「退職金は2,000万円から3,000万円。」
「社宅の家賃は月3,000円。」
```

断定します。

数字を使えるときは必ず使う。「約」「ほぼ」を使ってもいいですが、数字そのものを出すこと。

ただし当然ながら、事実確認が先です。Webで確認した数字しか使わない。曖昧な表現は「調べていない」か「自信がない」かのどちらかなので、使う前に必ず調べる。

---

## 落とし穴③：同じ機能のシーンを繰り返す

### 幸子チャンネルでよく起きる問題

幸子チャンネルのドラマ系の台本を書いていると、無意識に同じ機能のシーンが複数出てくることがあります。

例えば、「幸子が娘（美咲）からアドバイスをもらう」シーン。これが10分の動画に3回入っていたことがありました。

```
・スーパーの休憩室で美咲に相談する（2分頃）
・帰宅後に美咲から電話でアドバイスをもらう（6分頃）
・翌日また美咲に話を聞いてもらう（9分頃）
```

視聴者は2回目で「また同じ展開か」と感じます。3回目は完全に飽きています。

---

### なぜ繰り返してしまうのか

これ、台本を書いていると自然に起きます。

「幸子を助けてくれるシーンが必要」→「美咲を出す」→「また助けが必要」→「また美咲を出す」という流れが無意識に起きる。

各シーンを個別に見ると機能しているので、全体を通して読まないと気づきにくい。

---

### 解決策：キャラクターに「1回だけ使える役割」を与える

各キャラクターが物語の中で「何のために存在するか」を決めておいて、その機能を使うのは1回だけにする。

```
美咲（娘）：知恵・気づきを与える役 → 1回だけ使う
正夫（夫）：感情的な拠り所・笑いの役 → 1回だけ使う
田中さん：嫌味・壁になる役 → 繰り返し使ってもいいが同じパターンは避ける
```

台本を書いた後、全体を通して読んで「同じ機能のシーンが2回以上ないか」を確認する。あれば削るか、違う機能に変える。

---

## 落とし穴④：説明セリフが多すぎる

### 「セリフで説明する」の罠

台本を書くとき、登場人物に「自分の感情を説明させる」パターンをよくやってしまいます。

```
正夫（夫）：「お前はいつも頑張っているな。それが人間としての強さというものだ。」
```

これは「説明セリフ」です。正夫が幸子に向かって「あなたは強い人間だ」と言葉で説明している。

視聴者はこれを聞いたとき「言わされてる感」を感じます。現実の人間はこんなに上手い言葉を言わない。

---

### 解決策：説明より場面で見せる

「幸子が強い人間だ」と伝えたいなら、強い行動を見せる。

```
説明：正夫が「お前は強い」と言う

場面：幸子が一人でレジに立ち続けて、終業後に足をさすりながら「明日もあるからな」とつぶやく
```

後者の方が、セリフなしで幸子の強さが伝わります。

正夫のセリフは短くリズムで入れるのが正解です。

```
正夫：「太ったんじゃないか」
幸子：「太ってないわ」
正夫：「じゃあ気にするな」
```

長い洞察より短いテンポ。これだけで正夫のキャラクターと夫婦の関係性が伝わります。

---

## 転職チャンネルと幸子チャンネルの台本の違い

最後に、ジャンルによって台本の書き方がどう変わるかを整理します。

**転職チャンネル（情報系ショート）：**
```
目的：「この会社で働くとどんな感じか」を60秒で伝える
構成：逆説フック→具体的な数字の羅列→結論
文体：断定。曖昧な表現は使わない
登場人物：主人公（田中悠斗）が実態を体験する様子
```

**幸子チャンネル（ドラマ系ロング）：**
```
目的：「幸子の物語」に感情移入させる。10分見続けてもらう
構成：日常→問題発生→葛藤→解決（または次回へ続く）
文体：セリフ中心。感情の変化を見せる
登場人物：複数。各キャラが固有の役割を持つ
```

根本的に性格が違います。

転職チャンネルは「情報を密度高く詰め込む」台本が正解で、幸子チャンネルは「感情の流れを切らさない」台本が正解。同じアニメ動画でも、台本の書き方は全然違います。

---

## まとめ

1. **冒頭は逆説・矛盾から始める**
   「なぜ？」という疑問を作ることで視聴者を引き込む

2. **数字は断定する**
   曖昧な表現は使わない。使う前に事実確認をする

3. **同じ機能のシーンは1回まで**
   キャラクターに役割を与え、同じ機能を繰り返さない

4. **説明より場面で見せる**
   言葉で説明するより、行動・場面で伝える

5. **ジャンルによって台本の書き方が全然違う**
   情報系は密度を上げる、ドラマ系は感情の流れを守る
