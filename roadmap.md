# 判断不要リスト（迷ったらこれを見る）

**「今何をすべきかわからない」→ このリストの上から順番にやる**

1. 今週の台本テーマをClaudeに送ったか？ → まだなら今すぐ送る
2. Claudeから台本ドラフトが届いているか？ → あれば今日中に読む
3. タイアップの返信が来ているか？ → あれば今日中にClaudeに転送
4. 動画編集が止まっていないか？ → 木曜に必ず触る
5. 上記4つ全部OK → 休んでいい
